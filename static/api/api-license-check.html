License校验

商户试用期间或购买服务后，因特殊原因造成许可证校验失效时，申请许可license的重新校验 ，license是商户各类业务操作的凭证Token， 请妥善保管许可信息。 产品试用地址 产品demo。

如果你需要更多了解产品信息，请拨打公司服务热线。

属性 描述
id string 用户 ID ，64 位以内，首字母必须是英文、数字或者下划线。
object string 值为 "user"。
app string 对应 app 对象的 id。

示例对象

{
  "id": "test_user_003",
  "object": "user",
  "app": "app_1Gqj58ynP0mHeX1q"
}
